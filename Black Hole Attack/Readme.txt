This application has the standard protocol plus a simulation of a Black Hole attack. In this attack, malicious drones keep sending messages normally, but the neighbor list in every message will be empty. Every message sent by malicious nodes will not update information of its neighboring nodes, which will harm neighbor conscience of its neighbors.
  

Folder "Files Ns3" has all files to be pasted inside the ns3 folder. It has its own readme file!

"Python Scripts" has all scripts used to summarize informations from the simulations. It also has its own readme file!
